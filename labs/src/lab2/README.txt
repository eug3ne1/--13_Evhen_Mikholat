Для данної лабораторної стоврюється два класи
Main:
    - реалізація вводу даних за допомогою класу java.util.Scanner;
    - передача даних у якості параметрів на вхід методу MatrOP(int rows,int colmns, short[][]matrix1, short[][]matrix2)
MatrixCalc:
    - створення полів resultMatrix та columnAverages, необхідних для подальших операцій
    - обрахунок виключного АБО для двох заданих матриць
    - на основі сформованої матриці на попердньому кроці рахуємо середнє арифметичне кожного стовпця
    - виводимо отримані дані

Елементи матриць мають тип даних short відповідно до варанту
